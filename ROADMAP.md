# ROADMAP - CalculateChart 개발 로드맵

## P0 — 필수 항목

### 안정성 & 배포
- [ ] 프로덕션 `VITE_API_BASE_URL` 환경변수 Vercel에서 확인/설정
- [ ] Render cold start 대응 강화 (현재 `/api/ping` 웜업만 있음)
- [ ] 종목 리스트 캐시 유효성 검증 (장 마감 후 신규 상장 반영)

### 핵심 기능 검증
- [x] 자동완성: fdr 실패 시 pykrx 폴백 + 에러/로딩 UI 피드백
- [x] 차트 끊김 방지: NaN/0값 필터링 (백엔드+프론트)
- [x] 예측 모델: 누적 가격 캡 + GARCH 안정화 + 변동성 레짐 댐프닝
- [x] 지지/저항선: 다중 레벨(3개) + 최근성 가중 + 현재가 근처 필터링
- [x] 탭 타이틀 "차트 분석"으로 변경
- [ ] 배포 후 전체 기능 E2E 검증 (자동완성, 예측, 지지/저항)

## P1 — 이후 항목

### 차트 기능 확장
- [ ] 보조지표 패널 분리 (RSI/MACD를 차트 아래 별도 영역으로)
- [ ] 주봉/월봉 지원 (현재 일봉만)
- [ ] 캔들 패턴 자동 인식 (해머, 도지, 장악형 등) + 마커 표시
- [ ] 피보나치 되돌림/확장 오버레이
- [ ] 예측 신뢰구간 시각화 (상한/하한 밴드로 표시)

### 분석 고도화
- [ ] 일목균형표 추가 (한국 차트 분석의 핵심)
- [ ] 업종별/테마별 분류 및 필터링
- [ ] 백테스트: 과거 예측 정확도 자체 검증

### UX 개선
- [ ] 모바일 반응형 최적화 (현재 기본만 지원)
- [ ] 즐겨찾기 종목 (localStorage)
- [ ] 차트 설정 저장 (이평선 on/off, 색상 등)
- [ ] 다크모드/라이트모드 토글 (현재 OS 설정만 따름)
