## [v0.6] - 2026-04-13

### 자동완성 & API 연결
- `searchStocks` 에러 무시 → 에러/로딩 상태를 UI에 전달하도록 변경
- 자동완성 드롭다운에 "검색 중…" / "검색 결과 없음" / 연결 오류 메시지 표시
- `fetchOhlcv`, `fetchPrediction`의 "Failed to Fetch" → 한국어 에러 메시지로 변환

### 예측 모델 개선
- 과거 일일 변화율 분석 기반 변동성 레짐 분류 (극단적/안정적/보통)
- 극단적 변동 종목 댐프닝 — drift ±0.2% 제한, 수익률 캡 5%, 시그마 축소
- 안정적 종목 최소 변동성 0.5%/일 보장 + 최소 캔들 크기 확대

### 인프라
- 종목 리스트(`_load_listing`) 1시간 메모리 캐시 적용
- 서버 startup 시 종목 리스트 사전 로드
- OHLCV NaN/0값 행 필터링 (백엔드 + 프론트엔드 양쪽)
